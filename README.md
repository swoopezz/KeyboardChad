# KeyboardChad — tui typing app



### Installation
Linux:
```bash
git clone https://github.com/swoopezz/KeyboardChad && cd KeyboardChad
./install.sh 
kbchad
```

### Build
Linux:
```bash
git clone https://github.com/swoopezz/KeyboardChad && cd KeyboardChad
cmake -S . -B build && cmake --build build 
build/kbchad
```
